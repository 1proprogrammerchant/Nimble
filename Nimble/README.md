# Nimble: Rollback protection for cloud storage

## Building `nimble`
Install [`rustup`](https://rustup.rs/)

Clone the repository:
```text
git clone https://github.com/srinathsetty/Nimble
cd Nimble/Nimble
```

To run tests:
```text
cargo test
```

To build `libspartan`:
```text
cargo build --release
